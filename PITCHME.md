@title[Pointers]
# @size[3.5rem]( Pointers )

Note:

- Symbols (values)
  - *: Underlying value of
  - &: Memory address of
- Symbols (functions/methods)
  - *: By reference
  - (nothing): By value
- Why use pointers instead of values
  - Able to "persist" the changes to the struct
  - Avoid copying the struct when method is called
